## 安装

我们推荐使用 npm 或 yarn 的方式进行安装，不仅可在开发环境轻松调试，也可放心地在生产环境打包部署使用，享受整个生态圈和工具链带来的诸多好处。

```shell
// 使用npm

npm install panda-framework -S

// 使用yarn

yarn add panda-framework
```

```
import Vue from 'vue'

// 全局引入
import Panda from 'panda-framework'
